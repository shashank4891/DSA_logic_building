# Pattern-16: Alpha-Ramp Pattern

**Problem Statement:** Given an integer N, print the following pattern.

**Examples:**

![pattern16](https://github.com/user-attachments/assets/be4d2178-0331-4a62-9680-e8cdd4da1e2b)

Here, N = 5.

**Example 1:**
- Input: N = 3
- Output:
```
A
B B
C C C
```

**Example 2:**
- Input: N = 6
- Output:
```
A 
B B
C C C
D D D D
E E E E E
F F F F F F
```

# Pattern-11: Binary Number Triangle Pattern

**Problem Statement:** Given an integer N, print the following pattern.

**Examples:**

![pattern11](https://github.com/user-attachments/assets/b248ea8b-6412-4480-a7e5-20f1e1f3beeb)

Here, N = 5.

**Example 1:**
- Input: N = 3
- Output:
```
1
01
101 
```

**Example 2:**
- Input: N = 6
- Output:
```
1
01
101
0101
10101
010101
```

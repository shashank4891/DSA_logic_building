# Pattern-18: Alpha-Triangle Pattern

**Problem Statement:** Given an integer N, print the following pattern.

**Examples:**

![pattern18](https://github.com/user-attachments/assets/2242941a-e0c1-4331-bf5a-58cfa46e1a49)

Here, N = 4.

**Example 1:**
- Input: N = 3
- Output:
```
C
B C
A B C
```

**Example 2:**
- Input: N = 6
- Output:
```
F
E F
D E F
C D E F
B C D E F
A B C D E F
```

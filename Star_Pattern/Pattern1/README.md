# Pattern-1: Rectangular Star Pattern

**Problem Statement:** Given an integer N, print the following pattern.

**Examples:**

![pattern1](https://github.com/user-attachments/assets/1108cc46-a286-421d-98b3-2cdba9c2c400)

**Example 1:**
- Input: N = 3
- Output:
```
* * *
* * *
* * *
```

**Example 2:**
- Input: N = 6
- Output:
```
* * * * * *
* * * * * *
* * * * * *
* * * * * *
* * * * * *
* * * * * *
```

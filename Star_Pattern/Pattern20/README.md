# Pattern-20: Symmetric-Butterfly Pattern

**Problem Statement:** Given an integer N, print the following pattern.

**Examples:**

![pattern20](https://github.com/user-attachments/assets/2815687a-f87f-48be-aaa2-6e31b7440e3f)

Here, N = 4.

**Example 1:**
- Input: N = 3
- Output:
```
*    *
**  **
******
**  **
*    *
```

**Example 2:**
- Input: N = 6
- Output:
```
*          *
**        **
***      ***
****    ****
*****  *****
************
*****  *****
****    ****
***      ***
**        **
*          *
```

# Pattern-10: Half Diamond Star Pattern

**Problem Statement:** Given an integer N, print the following pattern.

**Examples:**

![pattern10](https://github.com/user-attachments/assets/933576db-8690-4f35-a96b-77a4efe0d115)

Here, N = 5.

**Example 1:**
- Input: N = 3
- Output:
```
  *  
  **
  ***  
  **
  *     
```

**Example 2:**
- Input: N = 6
- Output:
```
     *
     **
     *** 
     ****
     *****
     ******  
     *****
     ****
     ***    
     **
     *
```

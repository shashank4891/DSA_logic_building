# Pattern-12: Number Crown Pattern

**Problem Statement:** Given an integer N, print the following pattern.

**Examples:**

![pattern12](https://github.com/user-attachments/assets/8eb09223-9757-4c23-9519-7abdfb8e5308)

Here, N = 5.

**Example 1:**
- Input: N = 3
- Output:
```
1    1
12  21
123321
```

**Example 2:**
- Input: N = 6
- Output:
```
1          1
12        21
12       321
1234    4321
12345  54321
123456654321
```
